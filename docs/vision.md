# BookstAI

BookstAI est mon studio personnel de création assisté.

BookstAI doit :

- me faire gagner du temps ;
- améliorer mes workflows de création ;
- apprendre mon style ;
- réutiliser mes anciens contenus ;
- servir de terrain d'apprentissage pour l'IA agentique ;
- rester simple et peu coûteux.

Human In The Loop partout.

Review

Miniature
Storyboard

L'IA produit une première proposition.
↓
Je corrige.
↓

↓
Je valide.
↓
Tous les fichiers nécessaires sont générés.

# Constitution BookstAI

## Principes

BookstAI est un studio personnel.

Le Human In The Loop est obligatoire.

La mémoire est la source de vérité.

Les agents sont spécialisés.

Un agent ne fait qu'une chose.

Les workflows ne contiennent pas de logique métier.

La logique appartient aux composants.

La mémoire ne peut jamais être modifiée automatiquement.

Les sorties sont toujours validées.

La simplicité est préférée à la complexité.

On évite les dépendances inutiles.

Chaque nouvelle feature doit respecter cette Constitution.
